Requisitos de Senha Forte

Para maior segurança, sua senha deve conter:

  • Pelo menos 8 caracteres
  • Pelo menos uma letra maiúscula
  • Pelo menos uma letra minúscula
  • Pelo menos um número
  • Pelo menos um caractere especial (!@#$%^&*(),.?":{}|<>)

Evite:

  • Sequências óbvias como 123456, abcdef, qwerty
  • Informações pessoais como datas de nascimento
  • Palavras comuns como "senha" ou "password"
